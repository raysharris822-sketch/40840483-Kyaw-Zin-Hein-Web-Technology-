ABOUT  SOUNDBOARD X

SoundBoard X is an interactive browser-based soundboard built as part of a Web Technologies coursework project. It lets you play, organise and record sounds across multiple categories — all inside a single modern web application with no plugins or downloads required.

⚡ KEY FEATURES

🎵

MULTI-CATEGORY SOUNDS

5 categories — Anime, Phrase, Laugh, Music and Games — each with up to 12 sounds.

🎙️

VOICE RECORDER

Record your own voice with a live waveform display and instant preview before saving.

📂

MY RECORDINGS

Save, rename, play back and download recordings. Stored with IndexedDB — no upload needed.

⭐

FAVOURITES SYSTEM

Star your favourite sounds and filter the board to show only favourites with one click.

🔍

LIVE SEARCH

Instantly filter sound buttons by name as you type in the search bar.

📊

AUDIO VISUALIZER

Real-time frequency bar visualizer using the Web Audio API that reacts to every sound.

⚡

PLAYBACK CONTROLS

Play, pause, stop, replay, loop and adjust playback speed from 0.5× up to 2×.

⌨️

KEYBOARD SHORTCUTS

Control playback hands-free with Space, arrow keys, R to replay and L to toggle loop.

✨

PARTICLE ANIMATIONS

Gold particle burst and ripple effects trigger on every sound button click.

🌊

ANIMATED BACKGROUND

Canvas-powered background with floating orbs, sine waves and a gold grid overlay.

🛠 BUILT WITH
